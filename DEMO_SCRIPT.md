## חלק 3 — הדגמת DB ב-MySQL Workbench (40 שניות)

פתח MySQL Workbench עם החיבור למעלה והרץ את השאילתות הבאות:

### שאילתה 1 — משתמש חדש שנרשם
```sql
SELECT userId, firstName, email, userRole, createDate
FROM users
ORDER BY userId DESC
LIMIT 3;
```
> מראה: המשתמש החדש שנרשם בדמו

### שאילתה 2 — מתכון חדש שנוצר
```sql
SELECT recipeId, name, mealType, calories, imageUrl
FROM recipes
ORDER BY recipeId DESC
LIMIT 3;
```
> מראה: המתכון "אורז עם ירקות" + URL של Cloudinary (res.cloudinary.com/...) שנוצר אוטומטית ע"י AI

### שאילתה 3 — הגדרות שעודכנו
```sql
SELECT u.email, s.displayName, s.language
FROM user_settings s
JOIN users u ON s.userId = u.userId
ORDER BY s.userId DESC
LIMIT 3;
```
> מראה: שם התצוגה החדש שנשמר ב-Settings

### שאילתה 4 — מועדפים (Many-to-Many)
```sql
SELECT u.email, r.name AS recipe_name, r.mealType
FROM user_favorite_recipes uf
JOIN users u ON uf.userId = u.userId
JOIN recipes r ON uf.recipeId = r.recipeId
ORDER BY uf.createdAt DESC
LIMIT 5;
```
> מראה: המתכונים שסומנו בלייק — טבלת junction עם userId + recipeId

### שאילתה 5 — הודעות WebSocket שנשמרו
```sql
SELECT m.content, m.senderRole, u.email, m.createdAt
FROM support_messages m
JOIN users u ON m.userId = u.userId
ORDER BY m.createdAt DESC
LIMIT 3;
```
> מראה: ההודעות מהצ'אט עם התזונאי
